File structure for the Summer at I2BL, Microneedle Array Project:

1. Day 1-4 Outputs and SINGLES
--These are our dataset folders containing both our most recent multi-analyte dataset (containing approx. 84 conditions across 4 days, Day 1-4) and our singles data, which contains three folders for AA, DA, and UA.
2. src
--This folder contains every script written, and is approximately organized by function.
3. results
--This is an ongoing record of results allowing us to make visual comparisons, as this work is largely visual. Currently, a majority of these documents are in PDF format.
